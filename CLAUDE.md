# CLAUDE.md

The canonical repository instructions are in `AGENTS.md`. Read that file completely before planning, editing, reviewing, or running commands in this repository.

Then read the narrowest relevant project documentation:

- `docs/ARCHITECTURE.md` for accepted phase-1 decisions
- `docs/REFERENCE.md` for the operational overview
- `docs/API.md` for HTTP and external integration contracts, when present
- `docs/DATABASE.md` for D1 schema and state invariants, when present
- `docs/VERIFICATION.md` for verification requirements
- `docs/CONTRIBUTING.md` for Git and pull-request conventions

Instructions in `.claude/commands/` and `.claude/skills/` supplement `AGENTS.md`; they never override it or executable code and tests.
